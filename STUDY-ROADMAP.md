## ğŸ“ Study Tips

### Understanding vs Memorization
- **Understand WHY** services exist
- **Understand WHEN** to use each service
- **Understand HOW** services integrate
- Don't just memorize feature lists

### Service Comparisons
Create comparison tables for:
- EC2 vs Lambda vs Fargate
- RDS vs DynamoDB vs Aurora
- S3 vs EBS vs EFS
- ALB vs NLB vs GWLB
- CloudWatch vs CloudTrail vs Config

### Common Patterns Recognition
- **High Availability** = Multi-AZ + Auto Scaling + ELB
- **Cost Optimization** = Spot/Reserved + Auto Scaling + Lifecycle
- **Security** = Encryption + Private Subnets + IAM Roles
- **Performance** = Caching + Read Replicas + CDN
